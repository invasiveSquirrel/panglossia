# Polyglossia

Polyglossia is an AI-powered conversational language learning tool. It allows you to engage in natural dialogues with an AI tutor while receiving real-time linguistic support, grammar corrections, and vocabulary expansion.

## Key Features
- **Conversational Learning**: Chat with an AI tutor in your target language.
- **Dynamic Vocabulary Extraction**: Important words from your chat are saved into Markdown files.
- **Grammar Feedback**: Real-time corrections and grammatical explanations.
- **Multi-language Support**: Swedish, German, Finnish, Spanish, Portuguese, and Dutch.
- **Voice Integration**: High-quality TTS and STT via Google Cloud or local Piper fallback.

## Installation

### 1. Clone the repository
```bash
git clone https://github.com/invasiveSquirrel/polyglossia.git
cd polyglossia
```

### 2. Backend Setup
Create a virtual environment and install dependencies:
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Frontend Setup
```bash
cd ../frontend
npm install
```

## Running the Application

### The Easy Way (Linux)
Run the provided start script from the root directory:
```bash
./start.sh
```
This script handles starting the FastAPI backend, the Vite dev server, and launching the Electron window.

## How to Use Polyglossia

### 1. Choose Your Language
On the startup screen, select the language you want to practice.

### 2. Engage in Conversation
Simply type in the chat box. The AI tutor will respond in the target language.
- **Corrections**: The AI will provide feedback on your grammar and spelling.
- **Vocabulary**: When the AI uses a new or important word, it will be added to your local vocabulary file (e.g., `swedish_vocab.md`).
- **Voice**: You can hear the AI speak and record your own responses.

### 3. Study Vocabulary in Wordhord
The Markdown files generated by Polyglossia can be imported into **Wordhord** for spaced repetition training.

## Customizing the AI Tutor (Prompts)

You can customize how the AI behaves by editing the prompt files in the `backend/prompts/` directory. There is a specific text file for each language (e.g., `swedish.txt`).

### To Change the Tutor's Personality:
1. Open the relevant file, such as `backend/prompts/swedish.txt`.
2. Edit the instructions. You can change:
   - **Difficulty Level**: Tell the AI to use A2 or B1 level vocabulary.
   - **Tone**: Make the tutor more formal, more friendly, or more critical.
   - **Focus**: Tell it to focus more on certain grammatical topics or specific vocabulary themes (e.g., travel, business).
   - **Extraction Logic**: Customize how it decides which words are "important" enough to extract for your vocabulary files.

### Example Prompt Modification:
*Original:* "You are a friendly language tutor. Help the user practice Swedish."
*Modified:* "You are a strict Swedish tutor focusing on professional business language. Use B2 level vocabulary and correct every single grammatical error."

## Configuration
- **Ollama**: Polyglossia requires a local Ollama instance running the model specified in your environment (default: `gemma2:9b`).
- **Google Cloud**: Place your `google-credentials.json` in the root and set `GOOGLE_APPLICATION_CREDENTIALS` to enable high-quality voice features.
